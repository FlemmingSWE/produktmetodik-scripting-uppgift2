Produktvision: The CLI Escape Room
"Att genom en interaktiv spelupplevelse i PowerShell förvandla IT-säkerhetsutbildning från passiv information 
till praktisk färdighet. Genom realistiska utmaningar, direkt feedback och ett system där rätt svar belönas 
med pusselbitar för att låsa upp nästa rum, säkerställer vi att medarbetare bygger ett starkt digitalt självförsvar 
– bit för bit."

Varför pusselbitarna är geniala för ADKAR:
Mätbart framsteg (Knowledge): Varje pusselbit blir en visuell bekräftelse i terminalen på att spelaren har förstått 
och klarat en specifik utmaning.

Motivation (Desire): Att samla bitarna för att fysiskt "låsa upp" dörren till nästa rum förstärker Escape Room-känslan
och drivkraften att fullfölja spelet.
